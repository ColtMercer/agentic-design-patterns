# Appendices B–G — Supporting Topics

[← Back to README](../README.md)

---

## Appendix B — AI Agentic Interactions: From GUI to Real World

Shows how agents move from text-only interfaces into GUIs and physical environments. Topics include screenshots, element recognition, DOM mapping, and multimodal perception for real-world interaction.

## Appendix C — Quick Overview of Agentic Frameworks

Surveys the ecosystem — LangChain, LangGraph, CrewAI, Google ADK — and compares them on extensibility, tool integration, and safety primitives.

## Appendix D — Building an Agent with AgentSpace

Walks through constructing an agent with tools, resources, planning modules, prompts, and runtime loops.

## Appendix E — AI Agents on the CLI

Covers agent design for command-line environments: intent parsing, command mapping, streaming output, and safe shell composition.

## Appendix F — Under the Hood: Agents' Reasoning Engines

Looks inside agent internals: planners, short-term vs. long-term memory, Chain-of-Thought, Self-Consistency, Tree of Thoughts, and structured validators.

## Appendix G — Coding Agents

Focuses on code-writing agents: prompt templates, test-driven synthesis, sandboxed execution, debugging workflows, and CI integration.

---

## Why These Appendices Matter

The appendices broaden the book from core patterns into implementation context: how to prompt well, which frameworks to use, how agents interact with GUIs and terminals, and what reasoning engines and coding agents look like under the hood. They're less about one standalone pattern and more about the practical environment where all 21 patterns actually get built.
